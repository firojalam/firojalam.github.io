The **Tanbih API Platform** provides API-based access to AI services for combating fake news and disinformation in Arabic. It brings together research-driven capabilities such as claim detection, factuality analysis, check-worthiness estimation, media bias detection, propaganda technique detection, and media framing identification.

The platform is intended to make these disinformation-analysis services easier to integrate into downstream tools, applications, and workflows through a single API hub.
